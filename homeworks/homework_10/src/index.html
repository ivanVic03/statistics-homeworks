Simulation of a Counting Process

This page simulates a counting process by approximating a continuous-time phenomenon with a discrete model. The simulation follows a two-stage approach:

  1. Rate (λ) Calculation: First, we estimate the average event rate λ based on a related model. Using the parameters $n$ (steps), $m$ (trials), and $p$ (probability), the page runs $m$ simulations of a $Binomial(n, p)$ process. The average number of successes from these trials is used as our calculated rate λ.
  2. Process Simulation: This calculated λ is then used to run the primary simulation. The time interval $T=1$ is divided into $n$ subintervals, and an event is generated in each with probability $p_{sub} = \lambda/n$. This simulates a $Binomial(n, \lambda/n)$ process, which serves as a discrete-time approximation of a Poisson process.

The simulation displays the key results and visualizes the cumulative event path as a step-chart over the $[0, 1]$ interval.
