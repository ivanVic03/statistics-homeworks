Simulation of a Counting Process

This page demonstrates the approximation of a continuous-time stochastic process using a discrete-time simulation. The simulation employs a two-stage logical process:

  1. Rate (λ) Calculation (Monte Carlo Method):

    First, we establish an underlying average event rate, λ. Instead of assuming this rate, we derive it from a base model defined by three parameters: $n$ (steps), $p$ (probability), and $m$ (trials).

    This base model is a $Binomial(n, p)$ process, representing a sequence of $n$ independent Bernoulli trials, each with a success probability $p$. The theoretical expected number of successes for any single simulation is $E[K] = n \times p$.

    To estimate this expected value experimentally, the page runs $m$ independent simulations (trials) of this $Binomial(n, p)$ process. By the Law of Large Numbers (LLN), as $m \to \infty$, the sample mean of these $m$ results will converge to the true expected value. We use this sample mean as our calculated rate:

    \( \lambda = \frac{1}{m} \sum_{i=1}^{m} K_i \approx E[K] = n \times p \)
  2. Process Simulation (Poisson Approximation):

    This calculated rate λ is then used to parameterize the final simulation. The time interval $T=1$ is divided into $n$ subintervals.

    We simulate a new counting process by generating an event in each subinterval with probability $p_{sub} = \lambda/n$. As demonstrated by our calculation in step 1, this $p_{sub}$ will be numerically very close to the original $p$, since: \( p_{sub} = \frac{\lambda}{n} \approx \frac{n \times p}{n} = p \) This confirms the coherence of the estimation model.

    This second simulation is a $Binomial(n, \lambda/n)$ process. For large $n$ and small $p_{sub}$ (conditions which our model satisfies), the Poisson Limit Theorem states that this Binomial distribution converges to a Poisson distribution with parameter λ:

    \( \lim_{n \to \infty} Bin(n, \lambda/n) \to Poisson(\lambda) \)

The simulation displays the key parameters of this approximation (λ, $K$, and $p_{sub}$) and visualizes a single realization of the cumulative event path as a step-chart over the $[0, 1]$ interval.
