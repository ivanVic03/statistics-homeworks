Online Mean

Home

Starting from the classical definition of the mean after n observations:

\[ \mu_n = \frac{1}{n}\sum_{i=1}^n x_i. \]

We can split the sum into the first n-1 observations and the last one:

\[ \mu_n = \frac{1}{n}\left(\sum_{i=1}^{n-1} x_i + x_n\right). \]

Since \[ \mu_{n-1} = \frac{1}{n-1}\sum_{i=1}^{n-1} x_i, \] we can rewrite:

\[ \mu_n = \frac{1}{n}\big((n-1)\mu_{n-1} + x_n\big). \]

Simplifying, we obtain the online mean update:

\[ \mu_n = \mu_{n-1} + \frac{x_n - \mu_{n-1}}{n}. \]

This formula means that each new observation corrects the previous mean by a small amount proportional to the difference between the new value and the old mean.

Computational Advantages

  • Each update requires only the previous mean and count \(n\).
  • No need to recompute the sum of all past observations.
  • Memory usage is constant â†’ O(1).
  • Efficient for streaming or continuously arriving data.
