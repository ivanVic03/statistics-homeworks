Starting from the population variance definition after n observations:

\[ \sigma_n^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \mu_n)^2, \qquad \text{where } \mu_n = \frac{1}{n}\sum_{i=1}^n x_i. \]

Define the running sum of squared deviations:

\[ M_{2,n} \equiv \sum_{i=1}^n (x_i - \mu_n)^2, \] so that \[ \sigma_n^2 = \frac{M_{2,n}}{n}. \]

To update incrementally when a new observation \(x_n\) arrives, we write:

\[ M_{2,n} = M_{2,n-1} + (x_n - \mu_{n-1})(x_n - \mu_n). \]

The mean is updated at the same time as:

\[ \mu_n = \mu_{n-1} + \frac{x_n - \mu_{n-1}}{n}. \]

Finally, the variance can be recovered as:

\[ \text{Population variance: } \sigma_n^2 = \frac{M_{2,n}}{n}, \qquad \text{Sample variance: } s_n^2 = \frac{M_{2,n}}{n-1},\ n>1. \]

Computational Advantages

  • Updates require only three scalars: \(n\), mean, and \(M_2\).
  • Constant-time updates → O(1) per observation.
  • Constant memory — past data are not stored.
  • Numerically stable; reduces round-off errors.
