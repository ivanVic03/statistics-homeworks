The Axiomatic Solution — Kolmogorov's Unifying Framework

Kolmogorov’s Contribution (1933)

Kolmogorov did not attempt to define what probability is, but rather how it must behave. His axiomatic system separates mathematical structure from philosophical interpretation, giving probability a rigorous foundation within measure theory.

The Probability Space \((\Omega, \mathcal{F}, P)\)

$\Omega$ (Omega) – Sample Space: This is the non-empty set of all possible elementary outcomes ($\omega$) of a random experiment. For example, for a die roll, $\Omega = \{1, 2, 3, 4, 5, 6\}$.

$\mathcal{F}$ (F) – Sigma-Algebra of Events: This is a family of subsets of $\Omega$ that defines which subsets are considered measurable "events," i.e., sets to which it is meaningful to assign a probability. $\mathcal{F}$ must satisfy specific mathematical properties.

$P$ (P) – Probability Measure: This is a function $P: \mathcal{F} \to [0,1]$ that assigns a real number (the probability) to each event $A$ in the sigma-algebra $\mathcal{F}$.

Kolmogorov’s Axioms

Axiom 1 (Non-negativity): \(P(A) \ge 0\).

Axiom 2 (Normalization): \(P(\Omega) = 1\).

Axiom 3 (\(\sigma\)-additivity):

\[ P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i) \qquad\text{for disjoint } A_i. \]

Resolving the Bertrand Paradox with the Axiomatic Approach

The axiomatic approach does not "resolve" the philosophical debate between objectivists and subjectivists. A frequentist and a Bayesian will assign $P$ differently (the former via limits of relative frequencies, the latter via subjective beliefs), yet both must adhere to the same axioms to perform consistent calculations.

In the case of Bertrand’s Paradox, Kolmogorov’s framework does not provide a single "correct" answer. Instead, it shows that the original problem was ill-posed. The inconsistency arose from the ambiguity of the phrase "at random." By using the axiomatic approach $(\Omega, \mathcal{F}, P)$, the analyst is required to define the probability measure $P$ explicitly.

The three previously obtained "solutions" ($1/3$, $1/2$, $1/4$) are not contradictory. They are simply the correct results of three different experiments, i.e., three different choices of the probability measure $P$, each formalizing a distinct mechanism for generating the chord:

$P = 1/3$: Corresponds to a uniform measure $P_1$ over the angles (or arc lengths).

$P = 1/2$: Corresponds to a uniform measure $P_2$ over the radial distance from the center.

$P = 1/4$: Corresponds to a uniform measure $P_3$ over the area of the circle.

The axiomatization shifts the problem from an ambiguous philosophical question ("what is random?") to rigorous mathematical modeling ("which probability measure $P$ are we using?").
