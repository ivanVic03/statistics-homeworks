Formal Derivations from Kolmogorov’s Axioms

Preliminary Results

To provide the subadditivity and the inclusion-exclusion principle proofs, it's important to keep in mind the following properties:

  • Empty Set: \(P(\emptyset)=0\).
  • Complement: \(P(A^c)=1-P(A)\).
  • Monotonicity: If \(A\subseteq B\) then \(P(A)\le P(B)\).

Subadditivity Property (Boole’s Inequality)

Because events are sets, we use standard set-theoretic notation to describe their combinations. We write the intersection \(A \cap B\) to denote the occurrence of both events \(A\) and \(B\), and the union \(A \cup B\) to denote the occurrence of either event (or both).

For example, suppose we roll two dice. If \(A\) is the event that the first die shows a 1, and \(B\) is the event that the second die shows a 1, then:

  • \(A \cap B\) is the event that both dice show 1.
  • \(A \cup B\) is the event that at least one of the two dice shows 1.

We write the set difference \(A - B\) to denote the event that occurs in \(A\) but not in \(B\). In the same dice example, \(A - B\) consists of all outcomes where the first die is a 1 and the second die is not.

We denote the complement of an event \(A\) by \(\overline{A}\) as shorthand for \(\Omega - A\). For example, if \(E\) is the event of obtaining an even number when rolling a die, then \(\overline{A}\) is the event of obtaining an odd number.

For any two events \(A\) and \(B\):

\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]

Proof.

We decompose \(A \cup B\) into three disjoint events:

\[ A \cup B = (A - (A \cap B)) \;\cup\; (B - (A \cap B)) \;\cup\; (A \cap B). \]

(This corresponds to the logic of the Venn diagram.)

A more direct approach:

We decompose \(A \cup B\) into two disjoint events:

\[ A \cup B = A \;\cup\; (B - (A \cap B)). \]

By Axiom 3:

\[ P(A \cup B) = P(A) + P(B - (A \cap B)). \tag{1} \]

Now decompose \(B\) into two disjoint events:

\[ B = (B - (A \cap B)) \;\cup\; (A \cap B). \]

By Axiom 3 again:

\[ P(B) = P(B - (A \cap B)) + P(A \cap B). \tag{2} \]

Solve Equation (2) for \(P(B - (A \cap B))\):

\[ P(B - (A \cap B)) = P(B) - P(A \cap B). \]

Substitute this into Equation (1):

\[ P(A \cup B) = P(A) + P(B) - P(A \cap B). \]

Inclusion–Exclusion Principle

For any finite collection of events \(A_1, \dots, A_n\):

\[ P\!\left(\bigcup_{i=1}^{n} A_i\right) = \sum_{i} P(A_i) - \sum_{i} P(A_i \cap A_j) + \sum_{i} P(A_i \cap A_j \cap A_k) - \dots + (-1)^{n+1} P(A_1 \cap \dots \cap A_n). \] for \(i\) < \(j\) < \(k\)

Proof (by Mathematical Induction)

Base Case (\(n=2\)): See the previous proof.

Induction Step.

Inductive Hypothesis: Assume the PIE formula holds for \(n-1\) events.

Goal: Prove it holds for \(n\) events.

Define a new event \[ B = \bigcup_{i=1}^{n-1} A_i. \] Then the full union can be written as \[ \bigcup_{i=1}^n A_i = B \cup A_n. \]

Apply the \(n=2\) case to \(B \cup A_n\):

\[ P(B \cup A_n) = P(B) + P(A_n) - P(B \cap A_n). \tag{3} \]

Analyse the two terms:

• \(P(B)\): \[ P(B) = P\!\left(\bigcup_{i=1}^{n-1} A_i\right), \] and by the inductive hypothesis we may expand this using the PIE formula for \(n-1\) events.

• \(P(B \cap A_n)\):

\[ P(B \cap A_n) = P\!\left(\left(\bigcup_{i=1}^{n-1} A_i\right) \cap A_n\right) = P\!\left(\bigcup_{i=1}^{n-1} (A_i \cap A_n)\right) \]

By distributivity of intersection across unions. Again, this is a union of \(n-1\) events, so by the inductive hypothesis we may expand it using the PIE formula (this time applied to the events \(A_i \cap A_n\)).

Conclusion. Substituting the PIE expansions for \(P(B)\) and \(P(B \cap A_n)\) into Equation (3), all terms combine algebraically to yield exactly the full Inclusion–Exclusion formula for \(n\) events. Every term in \(\sum_{i} \, ... \,\), for \(i\) < \(j\) < \(k\), appears with the correct sign because it arises from either the expansion of \(P(B)\) or the expansion of \(P(B \cap A_n)\).
